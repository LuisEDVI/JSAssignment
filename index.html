Solução dos Exercícios

1 - Escreva um programa que mostre a soma de todos os divisores de um dado número X. Um divisor é qualquer número menor que X cujo resto da divisão é igual a 0.
Exemplo: Para X = 8, o resultado é 7, pois 7 = 1 + 2 + 4.

OBS: Use a função "prompt()" para utilizar inputs, para mostrar o resultado use o InnerText do elemento com ID: "result1"

2 – Considere dois pontos no espaço 2D com as seguintes coordenadas: A(x 1 , y 1 ) e B(x 2 ,y 2 ). A distância euclidiana D, entre estes pontos é dada pela seguinte fórmula:
D = √((x2 - x1)² + (y2 - y1)²)
Formula um programa que calcule tal distância para coordenadas de dois pontos informados pelo usuário.

OBS: Use a função "prompt()" para utilizar inputs, para mostrar o resultado use o InnerText do elemento com ID: "result2"

3 - Dada uma frase de entrada por um usuário, faça um programa que conte a ocorrência de cada palavra nesta frase. O programa não deve distinguir letras maiúsculas e minúsculas. Exemplo: “Faça chuva ou faça sol.”
Resultado: faça (2), chuva (1), ou, (1), sol (1)

OBS: Use a função "prompt()" para utilizar inputs, para mostrar o resultado use o InnerText do elemento com ID: "result3"

4 – Faça um programa que leia uma string de números como entrada de um usuário que representa uma matriz e diga qual dimensão dela (quantidade de linhas e colunas). As linhas da matriz são separadas por um caractere de ponto e vírgula ‘;’e as colunas são separadas por um espaço vazio ‘ ’, como no exemplo:
Entrada: 1 4 6 8; 3 5 6 7; 1 3 2 4
Resultado: Essa é uma matriz de dimensão 3 x 4.

OBS: Use a função "prompt()" para utilizar inputs, para mostrar o resultado use o InnerText do elemento com ID: "result4"

5 – Dado um número inteiro positivo N como entrada de um usuário, escreva um programa que mostre os primeiros N números da sequência de Fibonacci. (A sequência de Fibonacci é iniciada por 0 e 1, e o próximo número é sempre a soma dos dois últimos). Exemplo:
Entrada: 8
Resultado: 0, 1, 1, 2, 3, 5, 8, 13

OBS: Use a função "prompt()" para utilizar inputs, para mostrar o resultado use o InnerText do elemento com ID: "result5"
